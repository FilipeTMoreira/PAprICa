# PAprICa
Code for PAprICa IoT devices

Para mais informações sobre o projeto PAprICa - Potenciar Aprendizagens com a Internet das Coisas consulte o website paprica.web.ua.pt
onde encontrará toda a informação, nomeadamente os guiões de construção dos dispositivos IoT utilizados no projeto.

O dispositivo 1 (device 1) permite a obtenção de dados de humidade e temperatura do ar (com recurso ao sensor DHT22) e visualização dos mesmos na plataforma Thingsboard.

O dispostivo 2 (device 2) permite a obtenção de dados de pressão atmosférica, humidade e temperatura do ar (com recurso ao sensor BME280) e visualização dos mesmos na plataforma Thingsboard.

O dispostivo 3 (device 3) permite a obtenção de dados de pressão atmosférica, humidade e temperatura do ar (com recurso ao sensor BME280), humidade e pH do solo, índice UV e presença de fumos, sendo utilizada também a plataforma Thingsboard.
